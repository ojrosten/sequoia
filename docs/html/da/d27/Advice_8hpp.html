Detailed Description

Utilities for the advice framework, which provides hints for certain failures.

When a check fails, there may be some instances where the nature of the failure deserves comment. Indeed, such comments may provide a useful clue as to the origin of the failure, particularly when it is subtle. The advice framework facilitates this, allowing value-dependent comments to be generated in the instance of failure. This is achieved by passing an instance of tutor to the appropriate check call.
