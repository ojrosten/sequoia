Detailed Description

Implementation for a partitioned sequence of edges, which represents a graph's connectivity.

Graphs consist of edges and nodes. If the nodes do not possess a weight, they may be treated entirely implicitly. A partitioned sequence of edges, where each partition corresponds to a node and each edges records the other node to which it attaches, suffices to competely specify the graph. This data structure will be referred to as the connectivity. For graphs with weighted nodes, the same connectivity data structure may be used, but now in conjunction with a separate structure to record the node weights.
