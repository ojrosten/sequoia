template<class T>
struct sequoia::testing::value_tester< T * >

Compares instance of pointers.

Testing equality is performed via binary_comparison, and so does not require this specialization of value_tester.

The test(equivalence_check_t,...) function checks whether the pointers either both point to something or both point to nullptr, reporting a failure if this is not the case. If both pointers are not null, a check is dispatched to test the bound type. This is done using the strongest available check.
