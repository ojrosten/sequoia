template<class Compare, class T, class = std::void_t<>>
struct sequoia::testing::reports_for_type< Compare, T, class >

Reflection for wether an object of type Compare defines a function, report, which accepts two instance of a type, T.

Generally, when writing a new comparison class, it is desirable to provide detailed information in the case that the comparison fails. If this is done via a function, report, it will be automatically used by dispatch_check
