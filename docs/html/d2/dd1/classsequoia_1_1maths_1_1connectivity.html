Detailed Description

template<directed_flavour Directedness, class EdgeTraits, class WeightMaker>
class sequoia::maths::connectivity< Directedness, EdgeTraits, WeightMaker >

Graph connectivity, used as a building block for concrete graphs.

This class is flexible, allowing for representations of many different flavours of connectivity. It is designed for inheritance by concrete graphs; therefore various methods, including the destructor, are protected. Both static and dynamic graphs are supported; for the purposes of the latter, the relevant protected methods of connectivity are allocator-aware.
