Detailed Description

template<movable_comparable T, alloc_getter< T > Getter>
requires scoped_alloc<std::invoke_result_t<Getter, T>>
class sequoia::testing::allocation_info< T, Getter >

A specialization of allocation_info appropriate for std::scoped_allocator_adaptor.

The essential difference to the primary template is that multiple sets of predictions must be supplied, one for each level within the scoped_allocator_adaptor.
