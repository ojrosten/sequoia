template<class T>
class sequoia::ownership::spawner< T >

Class template providing a make method, which constructs an instance of T, wrapped in a proxy.

The proxy used is sequoia::utilities::uniform_wrapper<T>. The latter is understood to define the interface for other analogous proxies appearing in this file.
