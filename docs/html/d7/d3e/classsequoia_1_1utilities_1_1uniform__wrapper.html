Detailed Description

template<class T>
class sequoia::utilities::uniform_wrapper< T >

A wrapper which allows for getting, setting and mutation of its stored value.

This wrapper is designed for use alongside classes which expose proxies to the underlying data of interest. Throughout sequoia, such proxies should have the same get/set/mutate interface as uniform_wrapper. Thus the uniform_wrapper allows for a homogeneous treatment of families of classes, some of which must necessarily use proxies but all of which are desired to have the same semantics in terms of getting/setting/mutating the underlying data.
