Detailed Description

template<class Container, class... Allocators, class Predictions>
class sequoia::testing::basic_allocation_info< Container, std::scoped_allocator_adaptor< Allocators... >, Predictions >

A specialization of basic_allocation_info appropriate for std::scoped_allocator_adaptor.

The essential difference to the primary template is that multiple sets of predictions must be supplied, one for each level within the scoped_allocator_adaptor.
